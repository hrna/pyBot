<h2>IRC Bot written in Python.</h2>

pyBot Requirements:

Python 3.x

BeautifulSoup 4

Lxml or html5lib

python-dateutil

pystemmer (for Cobe)

In a case where you dont have the system administrators rights you may use easy install for these plugins
for example: 
easy_install3 --user pystemmer
easy_install3 --user python-dateutil
etc.. etc..
